# 📱 Laundry App (React Native - Android)

A modern **Laundry Management Mobile Application** built using React Native for Android devices.

This app allows users to view washing/pressing rates, place laundry orders, track order status, and provides a complete admin panel for managing services and orders.

---

# 🚀 Features

## 👤 User Side

- 📋 View washing and pressing rates of clothes on home screen
- 🧺 Select clothes and enter quantity for washing/pressing
- 🛒 Add items to cart
- 🧾 Place order with details:
  - Full Name
  - Phone Number
  - Address / Live Location
- 📦 Track order status using phone number
- 📱 Simple and user-friendly interface
- ⚡ Fast and offline-friendly UI (if applicable)

---

## 🔐 Admin Panel

- 🔑 Secure admin login (Email & Password)
- ➕ Add / manage clothing categories
- 💰 Set and update washing & pressing rates
- 📦 View all customer orders
- 🔄 Update order status:
  - Pending
  - In Progress
  - Completed / Delivered
- 🧠 Full control over app operations
